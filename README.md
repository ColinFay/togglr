# thinkR

an R and Rstudio wrapper for toggl Api.
<https://www.toggl.com/>

```R
library(togglr)
options(toggl_api_token = "XXXXXXXX")
toggl_start()
browseURL("https://www.toggl.com/app/timer")
```




## Installation



```R
# install.packages("devtools")
devtools::install_github("ThinkRstat/togglr")
```
