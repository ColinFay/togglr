# togglr

an R and Rstudio wrapper for toggl Api.
<https://www.toggl.com/>

```R
library(togglr)


toggl_start()
open_toggl_website() #browseURL("https://www.toggl.com/app/timer")



```




## Installation


```R
# install.packages("devtools")
devtools::install_github("ThinkR-open/togglr")



```
